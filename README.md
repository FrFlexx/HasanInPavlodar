# Hasan in Pavlodar

Локальный сайт для интерактива: участники сканируют QR-код, подключают микрофон телефона, а на большом экране две машины едут от уровня шума.

## Запуск

Самый простой вариант: откройте двойным кликом `start-race.bat`. Он создаст локальный HTTPS-сертификат и запустит сервер.

Или через терминал:

```powershell
cd C:\Users\FLEXX\Documents\Codex\2026-06-05\qr\outputs\noise-race
npm start
```

Откройте экран ведущего:

```text
http://localhost:3000
```

Для телефонов используйте адрес ноутбука в той же Wi-Fi сети. Сервер покажет его в терминале, например:

```text
http://192.168.1.25:3000/phone.html
```

## Микрофон на телефоне

Современные мобильные браузеры разрешают микрофон только на HTTPS-страницах. QR-код ведёт на `https://IP:3443/phone.html`. При первом открытии телефон может показать предупреждение о локальном сертификате, его нужно принять для тестового запуска. Для продакшн-мероприятия лучше использовать настоящий HTTPS-туннель, например Cloudflare Tunnel или ngrok.

## Как провести гонку

1. Запустите сервер и откройте `http://localhost:3000` на большом экране.
2. Участники сканируют QR-код, вводят имя и нажимают `Подключиться`.
3. Когда две машины подключены, экран автоматически перейдет на `race.html`.
4. На экране гонки нажмите `Старт`.
5. После финиша нажмите `Сброс`, чтобы вернуться на экран QR и подключить новую пару игроков.
